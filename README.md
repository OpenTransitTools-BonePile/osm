osm
===

Miscellaneous OSM processing tools

To get a running environment up, do the following:

0) git clone https://github.com/OpenTransitTools/osm.git
1) Install Python 2.7 (python 2.6, 3.0 might work, but not tested)
2) Install setup tools / easy install
3) Install buildout
4) cd osm
5) buildout
6) bin/python ott/abrev/osm_abbr_tester.py
